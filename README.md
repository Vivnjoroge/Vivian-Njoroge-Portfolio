
# Vivian Njoroge Portfolio

A modern, high-end portfolio website built with React, Tailwind CSS, and Framer Motion. Featuring cinematic animations, a custom particle background, and a responsive glassmorphism UI.

## üöÄ Assets Setup

To make the portfolio truly yours, follow these simple steps to add your personal assets:

### 1. Profile & Project Images
Create a folder named `img` in the root directory (same level as `index.html`).
Add the following files inside `img/`:
- `vivian.jpg` (Your main profile photo used in Hero & About sections)
- `inventory.jpg` (Project 1 image)
- `task-planner.jpg` (Project 2 image)
- `api-suite.jpg` (Project 3 image)
- `visualizer.jpg` (Project 4 image)

### 2. Resume / CV
Place your resume file in the root directory and name it:
- `cv.pdf`

The "Download CV" button in the Hero section is pre-configured to download this file.
